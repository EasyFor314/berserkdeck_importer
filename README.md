# Berserk Deck Importer

Расширение для браузера (Chrome / Edge), которое показывает **текстовый вид колоды** с [berserkdeck.ru](https://www.berserkdeck.ru) и копирует его в буфер обмена.

## Как пользоваться

1. Войдите на [berserkdeck.ru](https://www.berserkdeck.ru), если нужны приватные колоды.
2. Откройте страницу колоды, например:  
   https://www.berserkdeck.ru/decks/53458
3. Нажмите на иконку расширения в панели браузера.
4. В окне расширения нажмите **Скопировать**.

## Пример вывода

```
Герой:
Драккара, Дочь Огня

Основная колода:
2 Божественное вмешательство
2 Вскипятить кровь
3 День ярмарки
...

Дополнительная колода:
2 Выразить протест
2 Крематор
2 Сэмюэль
```

Карты в каждом разделе сортируются по алфавиту.

## Установка

> Chrome и Edge не устанавливают сторонние расширения «в один клик», как из магазина. При первой установке нужно один раз включить **режим разработчика** на странице `chrome://extensions` (в Edge — `edge://extensions`). Предупреждение о «непроверенном» расширении — нормальное поведение браузера.

### Из ZIP-архива

1. Скачайте `berserk-deck-importer.zip`.
2. Распакуйте архив в любую папку. Внутри должны лежать `manifest.json`, `popup.html` и остальные файлы.
3. Откройте `chrome://extensions` (в Edge — `edge://extensions`).
4. Включите **Режим разработчика**.
5. Нажмите **Загрузить распакованное расширение** и выберите распакованную папку.

### Из CRX-файла

1. Скачайте файл `.crx`.
2. Откройте `chrome://extensions`, включите **Режим разработчика**.
3. Перетащите `.crx` на страницу расширений и подтвердите установку.

Если Chrome откажется принять файл, установите расширение из ZIP — этот способ работает надёжнее.

После установки иконка расширения появится на панели инструментов браузера.

## Обновление

- **ZIP:** скачайте новый архив, замените файлы в папке расширения, на `chrome://extensions` нажмите **Обновить** (иконка ↻).
- **CRX:** скачайте новую версию и переустановите поверх старой.

## На каких страницах работает

- `https://www.berserkdeck.ru/decks/{id}`
- `https://berserkdeck.ru/decks/{id}`
- `https://www.berserkdeck.ru/decks/{id}/edit`

**Приватные колоды** — нужно быть авторизованным на berserkdeck.ru в той же вкладке. Для колод с кодом доступа параметр `?code=...` из адресной строки подхватывается автоматически.

## Ограничения

- Расширение работает только на страницах колод berserkdeck.ru
- Приватные колоды доступны только при входе в аккаунт на сайте
- Для недоступных или удалённых колод отображается сообщение об ошибке
- Состав колоды берётся с сайта в момент открытия окна расширения
